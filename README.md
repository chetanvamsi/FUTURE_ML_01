# 📊 Sales Forecasting Model

## 📌 Project Description
This project predicts future sales using Machine Learning (Linear Regression).

## 🚀 What I did
- Loaded sales data
- Converted date into month
- Trained Linear Regression model
- Predicted future sales

## 🔮 Predictions
- Month 7 → ~212
- Month 8 → ~231
- Month 9 → ~250
